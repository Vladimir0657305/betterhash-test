Coins

BTC

Bitcoin

The first cryptocurrency ever created, Bitcoin can be obtained by converting your mined coins.

ETH

Ethereum

We are using several Claymore Dual miners for Ethereum mining. The Ethash algorithm is available for both NVidia and AMD GPUs.

XMR

Monero

The Cryptonight algorithm works on both NVidia and AMD GPUs. You can mine Monero with your CPUs as well, with the XMRig miner.

ZEC

ZCash

ZCash can be mined with the Equihash algorithm with either EQMiner or EWBF. Each miner's profitability depends on your configuration.

ETC

Ethereum Classic

The Ethash algorithm works on both AMD and Nvidia GPUs. BetterHash provides Claymore miners for both NVidia and AMD.

BTG

Bitcoin Gold

You can mine for BTG regardless of your GPU chipset with the ZHash algorithm.

GRIN

Grin Coin

You can mine for GRIN with Nvidia chipsets with the Cuckatoo31 and the Cuckarood29 algorithms.

BEAM

Beam

The Beam coin is mined using the BeamHash III algorithm using both NVidia and AMD GPUs.

RVN

Ravencoin

Ravencoin has a long history of being one of our most profitable coins. You can mine Ravencoin with Nvidia and AMD GPUs, using the X16R algorithm.

FIRO

Firo

Firo can only be mined with Nvidia GPUs and it is mined with the MTP algorithm.

FLUX

Flux

Flux is a cryptocurrency that powers a system for app development, minable using the ZelHash algorithm.

ERG

Ergo

Ergo is a programmable blockchain able to host decentralized applications. It can be mined using both NVidia and AMD GPUs.

ETHW

Ethereum PoW

Ethereum PoW is a forked version of the Ethereum created after the Merge of the execution and consensus layers.
